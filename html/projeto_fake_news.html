Etapas do Projeto Fake News


Voltar para a Página Inicial

Carregamento e Preparação dos Dados

Carreguei dois datasets: um com notícias verdadeiras (True.csv) e outro com notícias falsas (Fake.csv).

Adicionei uma coluna de labels para identificar as notícias como 'REAL' ou 'FAKE'.

Combinei os datasets e embaralhei os dados para garantir a aleatoriedade.

Verifiquei a forma do DataFrame resultante, que contém um total de 44898 registros.

Divisão dos Dados

Separei o dataset em features (texto das notícias) e labels (classificações de FAKE ou REAL).

Dividi os dados em conjuntos de treino e teste, usando 70% dos dados para treino e 30% para teste, para avaliar o desempenho do modelo.

Transformação de Texto em Features

Utilizei o TfidfVectorizer para converter os textos das notícias em uma matriz TF-IDF, o que ajudou a quantificar a importância das palavras nos textos.

Treinamento do Modelo

Treinei um modelo de classificação Passive Aggressive usando os dados de treino transformados.

Avaliação do Modelo

Avaliei o desempenho do modelo calculando a acurácia e gerando uma matriz de confusão, que ilustra a performance do classificador na distinção entre notícias verdadeiras e falsas.

A acurácia obtida foi de 99.3%, indicando a eficácia do modelo.

Visualização

Utilizei Seaborn para visualizar a matriz de confusão normalizada, facilitando a interpretação dos resultados do modelo.
